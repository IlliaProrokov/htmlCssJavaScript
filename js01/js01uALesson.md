# Урок 01: Вступ до JavaScript

## Що таке JavaScript?

**JavaScript** — це мова програмування, яка робить веб-сторінки **інтерактивними** та **динамічними**. Тоді як HTML забезпечує структуру, а CSS — стиль, JavaScript забезпечує поведінку та інтерактивність.

### Що може робити JavaScript?

  - Реагувати на дії користувача (кліки, набір тексту, наведення курсору)
  - Динамічно змінювати вміст HTML
  - Змінювати стилі CSS "на льоту"
  - Перевіряти вхідні дані форм (валідація)
  - Створювати анімації та ефекти
  - Завантажувати дані без оновлення сторінки

### JavaScript працює у браузері

На відміну від деяких мов програмування, яким потрібне спеціальне програмне забезпечення, JavaScript виконується безпосередньо у вашому **веб-браузері**. Кожен сучасний браузер має вбудований рушій JavaScript.

-----

## Додавання JavaScript до вашого HTML

Існує три способи включення JavaScript у ваші веб-сторінки:

### Метод 1: Вбудований JavaScript (Inline JavaScript)

JavaScript можна писати безпосередньо в атрибутах HTML. **Не рекомендується для реальних проєктів**, але простий для розуміння:

```html
<button onclick="alert('Hello!')">Click Me</button>
```

У наведеному вище прикладі `onclick` є атрибутом події HTML, а `alert` — це функція JavaScript. Спробуйте код, скопіювавши та вставивши його у VS Code та відкривши в Live Server.

### Метод 2: Внутрішній JavaScript (Тег Script)

JavaScript можна писати всередині тегів `<script>` у вашому HTML-файлі:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First JavaScript</title>
</head>
<body>
    <h1>Hello World</h1>
    
    <script>
        // JavaScript code goes here
        alert('Welcome to JavaScript!');
    </script>
</body>
</html>
```

**Важливі примітки:**

  - Тег `<script>` зазвичай розміщують в кінці тегу `<body>`
  - Це гарантує, що HTML завантажиться до того, як почне виконуватися JavaScript
  - Для однорядкових коментарів у JavaScript використовують `//`

### Метод 3: Зовнішній файл JavaScript (Найкраща практика)

JavaScript можна написати в окремому файлі з розширенням **`.js`** і підключити його до вашого HTML:

**index.html:**

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <script src="script.js"></script>
</body>
</html>
```

**script.js:**

```javascript
alert('Hello from external file!');
```

**Чому зовнішні файли найкращі:**

  - Зберігає код організованим та розділеним
  - Дозволяє повторно використовувати той самий JavaScript на кількох сторінках
  - Легше підтримувати та оновлювати
  - Браузер може кешувати файл для швидшого завантаження

## Ваша перша програма JavaScript

Створімо повний приклад, який демонструє базовий вивід JavaScript:

```html
<!DOCTYPE html>
<html>
<head>
    <title>JavaScript Test</title>
</head>
<body>
    <h1>My First JavaScript</h1>
    
    <script>
        // This is a comment - JavaScript ignores this line (Це коментар - JavaScript ігнорує цей рядок)
        
        // Show a message box (alert) (Показати вікно повідомлення)
        alert('Hello, World!');
        
        // Write to the browser console (press F12 to see it) (Записати в консоль браузера (натисніть F12, щоб побачити))
        console.log('This appears in the console');
        
        // Write directly to the page (Записати безпосередньо на сторінку)
        document.write('JavaScript is working!');
    </script>
</body>
</html>
```

### Три способи вивести інформацію

1.  **`alert()`** - Показує **спливаюче вікно** повідомлення (popup message box)

      - Добре підходить для тестування та привернення уваги користувача
      - Блокує сторінку, доки користувач не натисне OK
      - Не часто використовується на реальних веб-сайтах

2.  **`console.log()`** - Пише в **консоль браузера**

      - **Найважливіше для навчання та налагодження\!**
      - Відкрийте Інструменти розробника (Developer Tools) (F12), щоб побачити вкладку Console (Консоль)
      - Не перериває досвід користувача

3.  **`document.write()`** - Пише безпосередньо на **HTML-сторінку**

      - Корисно лише для простого тестування
      - Може спричинити проблеми, якщо використовується після завантаження сторінки
      - Існують кращі методи (ми їх скоро вивчимо)

## Консоль браузера: Ваш найкращий друг

**Консоль** — це місце, де ви проведете багато часу, вивчаючи JavaScript.

### Як відкрити консоль:

  - **Windows/Linux:** Натисніть `F12` або `Ctrl + Shift + I`
  - **Mac:** Натисніть `Cmd + Option + I`
  - Потім натисніть на вкладку **"Console"**

### Що ви можете робити в консолі:

  - Бачити повідомлення від `console.log()`
  - Бачити повідомлення про помилки, коли щось йде не так
  - Вводити код JavaScript безпосередньо та негайно його тестувати
  - Налагоджувати свої програми (**Debug**)

### Спробуйте:

1.  Відкрийте будь-яку веб-сторінку
2.  Відкрийте консоль (F12)
3.  Введіть: `console.log("Hello from console!")`
4.  Натисніть Enter
5.  Ви побачите, як з'явиться ваше повідомлення\!

## Коментарі в JavaScript

**Коментарі** — це нотатки у вашому коді, які JavaScript ігнорує. Вони призначені для читання людьми.

```javascript
// This is a single-line comment (Це однорядковий коментар)

/*
This is a 
multi-line comment
It can span multiple lines
(Це багаторядковий коментар. Він може займати кілька рядків)
*/

// Use comments to explain what your code does (Використовуйте коментарі, щоб пояснити, що робить ваш код)
let age = 25;  // Store the user's age (Зберігаємо вік користувача)
```

**Коли використовувати коментарі:**

  - Пояснити складний або заплутаний код
  - Залишити нотатки для себе або інших розробників
  - Тимчасово вимкнути код під час тестування
  - Документувати, що роблять функції
